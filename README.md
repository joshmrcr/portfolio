# Brian Joshua Marcera - Portfolio

A fully responsive, modern minimalist single-page portfolio built to showcase my projects, technical skills, and experience as a Computer Engineer and Software Engineer.

## Features
- **Deep Dark Aesthetic:** Designed with a `#050505` background and vibrant neon glowing accents.
- **Micro-Animations:** Smooth scroll reveals utilizing `IntersectionObserver` and glassmorphism hover effects on project cards.
- **Fluid Responsiveness:** Built mobile-first using CSS Grid and Flexbox for seamless adaptation to all screen sizes.
- **Deploy Ready:** Includes a GitHub Actions workflow (`.github/workflows/deploy.yml`) for instant, continuous deployment to GitHub Pages.

## Core Sections
1. **Hero:** High-impact introduction and Call-to-Action.
2. **About:** Personal biography paired with a structured "Technical Toolkit" grid for skills.
3. **Projects:** Interactive showcases of recent work with dynamic links to live sites and source code.
4. **Contact:** Footer section with direct mail and social media profiles.

## Installation & Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/joshmrcr/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the Vite development server:
   ```bash
   npm run dev
   ```
   *The local server typically runs on `http://localhost:5173/`.*

4. Build for production:
   ```bash
   npm run build
   ```

## Deployment
This project is configured to automatically build and deploy to **GitHub Pages** whenever changes are pushed to the `main` branch, courtesy of the included GitHub Actions workflow. Ensure that GitHub Pages settings in your repository point to the `github-pages` environment generated by the action.
